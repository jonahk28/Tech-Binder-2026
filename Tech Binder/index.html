Drivetrain & Electronics Structure
DRV.02

Swerve Base & Frame

Low CG frame profile constructed for high impact absorption and immediate service access.

Frame Features

  • Drivetrain Platform: SDS MK4i swerve modules paired with Kraken X60 motors operating at L3+ high-velocity gear ratios.
  • The Brainpan: All key electronics are inverted on a split-pan design, protected externally by a 1/4" thick clear impact-grade polycarbonate lower shield plate.
  • Bellypan Design: Employs a non-pocketed, solid 1/8" aircraft-grade aluminum sheet to optimize mass down low and guarantee structural torsion rigidity.
  • Bottom-Out Battery Mount: Bottom-loading structural tray assembly with a single retention fastener for rapidly executing physical battery swaps under 15 seconds.
  • Inertial Processing: Driven by a low-latency CTRE Pigeon 2.0 IMU tracking heading parameters.

Ground Intake Mechanism
INT.03

Ground Intake & Pivot

High-speed active floor collection built to survive extreme collision environments.

Intake Assembly Facts

  • Ground Intake Roller: Powered by a Kraken X44 brushless motor using a 2:1 reduction ratio driving high-durometer 2" compliant wheels.
  • Ground Intake Pivot: High-torque actuation handled by a Kraken X44 integrated into a heavy-duty 50:1 MaxPlanetary gearbox for absolute positional rigidity.

Indexer
IDX.04

Conveyor Systems

Conveyor Specs

  • Roller Floor: Heavy-wall 1.25" outer diameter polycarbonate roller paths configured with custom Thriftybot dead axles driven via a Kraken X60 at a 2:1 belted reduction.
  • Indexer Bottom Belt: Actuated by a Falcon 500 motor utilizing a 2:1 reduction with a large 36t precision profile pulley wheel.
  • Indexer Top Belt: Managed by an isolated Kraken X44 motor running a 2:1 reduction and a compact 15t timing pulley configuration.
  • Anti-Jam Entry: Active auxiliary feeder stage right before the flywheel chamber powered by a space-saving Minion motor at a 36:24 reduction to eliminate piece binding.
  • Sensor Triggers: Employs a CANrange Time-of-Flight distance sensor to immediately identify item containment. Code routines instantly pause internal travel paths on detection.

Shooter
SHT.05

Launcher System

Shooter Configuration

  • Flywheel Power: Dual independent Kraken X60 motors operating in a 1:3 speed-increasing gear loop to deliver massive rotational energy to the flywheels.
  • Launch Wheels: Projects tracking targets accurately using highly predictable, non-expanding 4-inch outer diameter Stealth wheels.
  • Adjustable Hood: Powered by a precise Kraken X44 driving an inside 3D-printed curved 10DP rack and pinion module for instant, accurate angle shifts.
  • Initialization Homing: Automates sensor homing on boot-up by smoothly driving the mechanism to concrete physical stops to find hard zero coordinates.

Software Calculations
MTH.07

AimCalc & Targets

Real-time pose projection mathematics managing high-velocity on-the-fly execution.

Core Code Tasks

1. Virtual Target Leading

To compensate for chassis translation momentum while executing a shot on the move, the math routines determine a virtual target position ahead of the real-world frame goal. This calculates vector adjustments by crossing instant chassis speed with real-time flight metrics.

2. Grid Interpolation Maps

Utilizes multi-point distance lookups. When localized anywhere between designated landmarks, the tracking code applies multi-dimensional linear interpolations to extract target RPM states and exact hood positioning.

3. Obstacle Height Compensation

The logic automatically evaluates vector clearance paths against noted field defensive obstacles. If an obstacle compromises a linear shot trajectory, the code overrides standard profiles to evaluate a steep parabolic solution.
